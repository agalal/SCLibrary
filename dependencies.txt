To run the SCLibrary application you will need to download this software:

1. Git Shell
2. Node.js 				https://nodejs.org/en/download/
3. Neo4j database    	http://neo4j.com/
4. redis server
	windows:    https://github.com/MSOpenTech/redis/releases/tag/win-2.8.2400
	all others: http://redis.io/download

Once all of this is installed, download the project with this command
	git clone https://github.com/mimen/SCLibrary.git SCLibrary

Navigate to the directory and run these commands to install dependencies:
	bower install
	npm install

Launch the neo4j application to initialize a server. Navigate to the url given
to set it up. Remember your username and password. Default is neo4j/neo4j.

Sign up for an application on soundcloud to get an api key
	http://soundcloud.com/you/apps

Create a config folder in the main directory. Inside, create the file "local.js"
Copy and paste this code and fill in the necessary info.

___________________________________________________________________
	var config = {

		base_url: 'http://localhost:3000/',
		auth: { client_id: 'YOUR_CLIENT_ID',
				client_secret: 'YOUR_CLIENT_SECRET',
				redirect_uri: 'http://localhost:3000/home'
			  },
	  	session_secret : "thisisasecret69",
		neo4j_href : 'http://YOUR_USERNAME:YOUR_PASSWORD@localhost:7474'

	}

	module.exports = config;
___________________________________________________________________


Once this has all been setup, navigate to the directory SCLibrary
and run the command 'nodemon'. This will launch your node app and 
restart the app every time you make any changes.
